I'm trying to access my mailbox over the 'shfs' file system, which
uses 'ssh' to mount the remote system. 

When I open my Mailbox this way, I get the message:

 "Lock count exceeded, remove lock?"

This is not a helpful error message. I don't anything about locks
and don't think I should need to. At least, the implications of selecting 
"yes" or "no" should be explained. A "Cancel" option would be nice as well. 

I selected "No" and the mailbox seemed to open fine, but implications of this 
choice remain unclear.
